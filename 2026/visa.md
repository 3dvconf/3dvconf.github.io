**Visa Conditions**

To check if you require a valid visa to enter Canada and for more information on visa applications please refer to this [IRCC website](https://www.canada.ca/en/immigration-refugees-citizenship/services/visit-canada.html?outside).

If required, please apply for a visa as early as possible.
Current visa processing times are listed [here](https://www.canada.ca/en/immigration-refugees-citizenship/services/application/check-processing-times.html).

**Visa Invitation Letters** 

Information on requesting an invitation letter is provided in the registration process.
Please follow up with us as soon as you have registered for the conference.
